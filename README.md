# SaaS ROI Calculator

A standalone React application for calculating SaaS ROI, revenue impact, and retention analytics.

## Features

- Calculate revenue impact of customer churn
- Analyze expansion revenue effects
- Model growth scenarios
- Interactive charts and visualizations
- 12-month revenue forecasts

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This app is deployed to Vercel as a standalone application.

## URL

Production: https://www.saasfactor.co/tools/saas-roi-calculator

## Original HTML

The original HTML file has been preserved as `index.html` in the root folder.
The content needs to be converted into React components in the `src/` directory.
